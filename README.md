DevOps Internship Project

This repository contains the tasks completed during my DevOps Internship.
The project demonstrates the implementation of CI/CD pipelines, automated deployment, and code quality analysis using modern DevOps tools.


Tools & Technologies Used

* Git & GitHub
* GitHub Actions
* SonarCloud
* GitHub Pages
* YAML


Task 1 – Repository Setup

Created a GitHub repository and initialized the project structure.
Version control was implemented using Git, and the repository was organized to support CI/CD workflows.

Key Activities:

* Created repository
* Added project files
* Maintained commits using Git


Task 2 – Continuous Integration (CI) Pipeline

Implemented a CI pipeline using GitHub Actions to automate the build and workflow execution whenever code is pushed to the repository.

Key Activities:

* Created GitHub workflow file
* Configured automatic pipeline execution
* Verified successful workflow runs in GitHub Actions


Task 3 – Deployment using GitHub Pages

Configured GitHub Pages to deploy the project automatically from the repository.

Key Activities:

* Enabled GitHub Pages
* Configured deployment workflow
* Verified successful project deployment


Task 4 – Code Quality Analysis using SonarCloud

Integrated SonarCloud with GitHub Actions to perform automated static code analysis.

Key Activities:

* Connected GitHub repository to SonarCloud
* Generated authentication token
* Configured SonarCloud workflow
* Executed automated code scan through CI pipeline


Conclusion

This project demonstrates the integration of CI/CD pipelines, automated deployment, and code quality monitoring using modern DevOps practices.
